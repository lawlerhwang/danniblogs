# Blog Posts

Add your MDX blog posts here. Each post should be a `.mdx` file with frontmatter.

## Frontmatter Format

```mdx
---
title: Your Post Title
date: 2024-01-15
description: A brief description of your post
---

# Your Post Content

Write your blog post content here using Markdown syntax.
```

See `welcome.mdx` for an example blog post.
